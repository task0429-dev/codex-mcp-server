# KPI Board Spec

## Daily KPIs
- Lead volume
- Response rate
- Meeting rate
- Stage conversion

## Alert Logic
- Yellow: drift worse than yellow threshold versus 30-day baseline.
- Red: drift worse than red threshold or 3 consecutive yellow days.

## Owner / Status / Next Action
- Owner: Prime
- Status: In Progress
- Next Action: wire automated daily feed from CRM sync status stream.
